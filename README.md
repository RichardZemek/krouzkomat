# Informační systém pro správu školních kroužků "Kroužkomat"

## Krátký popis  
Webová aplikace pro správu školních kroužků, která umožňuje učitelům a žákům přehledně organizovat přihlášky, docházku a platby.  

## Cíl systému a cílová skupina  

**Cílová skupina:**  
- Vedoucí kroužků (učitelé, lektoři)  
- Žáci a jejich rodiče  
- Administrátor školy (správce systému)  

**Cíl systému:**  
Systém má usnadnit správu školních kroužků – od přihlašování žáků přes evidenci docházky až po sledování plateb.  
Nahradí ruční evidenci v Excelu a papírové seznamy, zajistí přehlednost, rychlou komunikaci a úsporu času.  

## Základní funkce  

### Vedoucí kroužku  
- Zobrazit seznam přihlášených žáků  
- Evidovat docházku a hodnocení  
- Přidávat nebo rušit lekce  
- Posílat zprávy přihlášeným žákům  

### Žák / Rodič  
- Prohlížet dostupné kroužky a přihlásit se  
- Vidět své přihlášky, docházku a platby  
- Odhlašovat se z kroužků nebo jednotlivých lekcí  

### Administrátor školy  
- Spravovat uživatele (učitele, žáky, rodiče)  
- Zakládat nové kroužky a přiřazovat jim vedoucí  
- Nastavovat školní rok a termíny plateb  
- Generovat přehledy o obsazenosti a příjmech  

## Spravovaná data (příklady)  
- **Uživatelé:** jméno, role, kontaktní údaje  
- **Kroužky:** název, popis, vedoucí, kapacita, cena  
- **Docházka:** datum, přítomnost, poznámky  
- **Platby:** žák, kroužek, částka, stav (zaplaceno/nezaplaceno)  
- **Zprávy:** odesílatel, příjemce, text, datum odeslání  

## Možné rozšíření do budoucna  
- Mobilní aplikace pro potvrzení docházky  
- Automatické notifikace o blížící se platbě  
- Integrace s elektronickou žákovskou knihou
